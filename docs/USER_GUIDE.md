## Introduction

The ECOWAS Summit TWG System is an AI-powered platform designed to support the coordination of six Technical Working Groups (TWGs) for the ECOWAS Economic Integration & Investment Summit 2026.

### What Can the System Do?

- **Automate Meeting Coordination**: Schedule meetings, send invitations, track attendance
- **Generate Documents**: Create agendas, minutes, policy papers automatically
- **Track Action Items**: Monitor tasks and send reminders
- **Manage Projects**: Curate and score investment proposals
- **Facilitate Collaboration**: Real-time updates and notifications

## Getting Started

### Logging In

1. Navigate to the system URL (provided by your administrator)
2. Enter your email address and password
3. Click "Login"

**First Time Login**:
- You'll receive a welcome email with your temporary password
- You'll be prompted to change your password on first login

**Forgot Password**:
- Click "Forgot Password" on login page
- Enter your email
- Follow the reset link sent to your email

### Your Dashboard

After logging in, you'll see the Dashboard with:
- Overview of your assigned TWG(s)

- Recent documents

## Working with Meetings

### Scheduling a Meeting

1. Click "Schedule Meeting" button
2. Fill in meeting details:
   - **Title**: Meeting subject
   - **Date & Time**: When it will occur
   - **Duration**: Length in minutes
   - **Participants**: Select from TWG members or add emails
   - **Agenda Items**: List topics to discuss
3. **Optional**: Attach pre-read documents
4. Click "Schedule"

**What Happens Next**:
- AI agent generates a formal agenda
- Invitations are emailed to all participants
- Calendar invites (.ics files) are attached
- Meeting appears on your calendar

### Preparing for a Meeting

**Before the meeting**:
- Review the agenda (emailed to you)
- Read any attached documents
- Submit agenda item suggestions (if applicable)

**Reminders**:
- You'll receive a reminder 24 hours before
- Another reminder 1 hour before (for virtual meetings)

### During a Meeting

**If using transcription** (virtual meetings):
- The system can join to record and transcribe
- Real-time notes are captured
- Action items are flagged automatically

**Manual Note-Taking**:
- Designate a note-taker
- Upload notes after the meeting

### After a Meeting

**Automated Process**:
1. AI agent drafts meeting minutes
2. Facilitator receives draft for review
3. Facilitator approves or edits
4. Final minutes are emailed to participants
5. Action items are added to tracker

### Viewing Meeting Details

Click any meeting to see:
- Full agenda
- Attendees and apologies
- Meeting minutes (if completed)
- Action items assigned
- Related documents

## Agent Chat Interface

### What is the Agent?

Each TWG has a dedicated AI agent that:
- Knows your TWG's context
- Accesses TWG documents
- Performs tasks automatically
- Learns from your inputs

### How to Use the Agent

**Location**: Find the agent chat in:
- TWG Workspace sidebar
- Floating chat button (bottom-right)

**Asking Questions**:
```
You: When is our next meeting?
Agent: Your next meeting is on Feb 15, 2026 at 10:00 AM GMT.

You: Summarize decisions from the last 3 meetings.
Agent: [Provides summary with sources]
```

**Giving Commands**:
```
You: Draft an agenda for the upcoming meeting.
Agent: I've prepared a draft agenda. [Shows preview]
      Would you like to review and approve?

You: Schedule a meeting for next Friday at 2pm.
Agent: I can help with that. Who should I invite?
```

**Approving Agent Actions**:
- Agent will request approval for important actions
- Review the proposed content/action
- Click "Approve" or "Edit" or "Reject"
- Agent executes after approval

### Agent Capabilities

**Meeting Management**:
- Schedule meetings
- Send reminders
- Draft agendas
- Generate minutes

**Document Tasks**:
- Draft policy papers
- Summarize documents
- Extract key points
- Create briefing notes

**Information Retrieval**:
- Answer questions about TWG work
- Find relevant documents
- Provide context from knowledge base

**Project Support**:
- Evaluate project proposals
- Generate project briefs
- Track project status

### Tips for Working with the Agent

**Be Specific**:
- ✅ "Draft a 2-page concept note on renewable energy targets"
- ❌ "Write something about energy"

**Provide Context**:
- Reference previous meetings or documents
- Specify desired format or length
- Mention any constraints

**Review Outputs**:
- Always review AI-generated content
- Check facts against source documents
- Edit for tone and accuracy
- The agent is a draft assistant, not a replacement for human judgment

## Project Pipeline

### Overview

The Project Pipeline (mainly for Resource Mobilization TWG) tracks:
- Investment proposals
- Bankable projects
- Deal room candidates
- Funding commitments

### Adding a Project

1. Go to Projects section
2. Click "Add Project"
3. Fill in project details:
   - **Name**: Project title
   - **Pillar**: Which TWG pillar it supports
   - **Description**: Brief overview
   - **Lead Country/Entity**: Who's proposing
   - **Investment Size**: Required funding
   - **Sector**: Industry category
4. **Optional**: Upload business case, feasibility study
5. Click "Submit"

### Project Scoring

Projects are scored on:
- **Strategic Alignment** (30%): Fits summit priorities
- **Readiness** (25%): Has necessary studies/approvals
- **Regional Impact** (20%): Benefits multiple countries
- **Financial Viability** (15%): Bankable business case
- **Innovation** (10%): Novel approach

**Score Display**:
- 0-40: Low priority
- 41-70: Medium priority
- 71-100: High priority (flagship candidate)

### Project Status

- **Submitted**: Initial submission
- **Under Review**: Being evaluated
- **Approved**: Selected for summit
- **Deal Room**: Presentation-ready
- **Closed**: Deal signed or rejected

### Viewing Project Details

Click any project to see:
- Full description
- Scoring breakdown
- Documents attached
- Status history
- Comments and notes

## Common Tasks

### How do I...

**...cancel a meeting?**
1. Go to meeting details
2. Click "Cancel Meeting"
3. Provide reason
4. System notifies all participants

**...reassign an action item?**
1. Go to Action Items
2. Click the item
3. Select new assignee
4. Add comment if needed
5. Save

**...export a report?**
1. Navigate to relevant section (Meetings, Projects, etc.)
2. Click "Export" button
3. Choose format (PDF, Excel, CSV)
4. Download file

**...invite a guest to a meeting?**
- Add their email in the participants field when scheduling
- They'll receive invitation even if not in system

## Troubleshooting

### I can't log in
- Verify email and password are correct
- Check if Caps Lock is on
- Try "Forgot Password" to reset
- Contact your system administrator

### I didn't receive a meeting invitation
- Check your spam/junk folder
- Verify your email in profile settings
- Ask facilitator to resend

### Documents won't upload
- Check file size (<10MB)
- Verify file format is supported
- Try a different browser
- Contact support

### Agent is not responding
- Check your internet connection
- Refresh the page
- Try a different command
- Report to administrator if persists
